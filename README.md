## Use

``` bash
# install dependencies
npm install

# run serve
# open http://127.0.0.1:8081/debugger-ui
react-native start

# run in android
react-native run-android

# run in ios
react-native run-ios
```

### use vscode

install react-native-tools

use vscode open file.

run debug (F5).
